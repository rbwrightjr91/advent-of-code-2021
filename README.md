# ReScript Project Template

The only official ReScript starter template.

## Installation

```sh
npm install
```

## Build

- Build: `npm run build`
- Clean: `npm run clean`
- Build & watch: `npm run start`

## Run

```sh
node src/<filename>.bs.js
```

or

```sh
npm run day-XX[:test]
```

where XX is 01-25
